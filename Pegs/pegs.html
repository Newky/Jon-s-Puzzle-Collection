Peg Solitaire

Remove all but one peg, finishing with the final peg in the middle of the board.

Pegs can be removed by jumping them, as in draughts. Only vertical and horizontal moves are allowed; diagonal moves are not allowed.

Feedback, particularly code improvements, welcome. Send it to jonrob DOT one AT googlemail DOT com.

Original inspiration for creating this game came from Simon Tatham's Portable Puzzle Collection. I hope to recreate more of these in Javascript + HTML5 canvas as I learn to program.

Code for detecting and correctly placing mouse clicks came from Dive Into HTML5 - an excellent book.
